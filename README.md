# mini.olhafair.com

Лендинг міні-курсу від Ольги Грицай. React + Vite + TypeScript + Tailwind CSS.

## Локальна розробка

```bash
npm install
npm run dev
```

## Деплой на Cloudflare Pages / Vercel / Netlify

| Параметр | Значення |
|---|---|
| Build command | `npm run build` |
| Build output directory | `dist` |
| Змінна оточення | `NODE_VERSION = 20` |
